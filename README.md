Court Hearing Scheduling System is a Java Web Application (Servlet + JSP + JDBC) used to add, store, and view court hearing schedules in a database.

 Main Objective

To build an online system where users can:

 Add a new hearing schedule

 View a specific hearing schedule

 View all scheduled hearings

All data is stored in a database table (HEARING_TB) and managed using Java backend layers.
